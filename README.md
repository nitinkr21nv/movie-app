# Movie Finder

A simple and interactive movie search web application built using React and Vite.

## Description

This project allows users to search for movies and view their details in a clean and user-friendly interface. Movies are displayed in a grid layout, and clicking on any movie opens a popup showing detailed information like title, year, genre, actors, and plot.

The app also shows some default movies when it loads and supports automatic searching while typing.

---

## Features

* Search movies in real-time
* Default movies shown on homepage
* Click on a movie to view details in popup modal
* Fast performance using Vite
* Clean and responsive UI

---

## Tech Stack

* React
* Vite
* OMDB API

---

## Installation & Setup

Clone the repository:

```bash
git clone https://github.com/nitinkr21nv/movie-app.git
cd movie-app
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

---

## Environment Variables

Create a `.env` file in the root folder and add:

```
VITE_API_KEY=your_api_key_here
```
